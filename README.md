# Deep-Learning-Based-COVID-19-Classification-Using-Chest-Radiography-Images
A deep learning model based on CNN and ResNet50 is used for automated COVID-19 detection from chest X-ray images. Transfer learning and fine-tuning improve accuracy and performance.

[COVID-19 Radiography Dataset](https://www.kaggle.com/datasets/preetviradiya/covid19-radiography-dataset)
